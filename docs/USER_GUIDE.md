# GW2 Progression — 用户指南

> **你的 GW2 成长决策引擎。**  
> 输入 API Key，立即了解账号价值、下一步行动、7 天成长路径。

---

## 🚀 快速开始（10 秒看到价值）

### 1. 获取 API Key

访问 [ArenaNet Applications](https://account.arena.net/applications)，创建新 Key：

| 权限 | 用途 |
|------|------|
| `account` | 账号名和世界 |
| `characters` | 装备与背包分析 |
| `inventories` | 银行、材料、共享背包 |
| `wallet` | 金币与货币 |

> ⭐ 推荐额外开启：`tradingpost`、`builds`、`progression`、`unlocks`

### 2. 粘贴并分析

将 Key 粘贴到首页，点击 **Analyze**。约 10 秒后你会看到：

```
💰 总价值 12,340g     🪙 钱包 500g
⚔ 最佳 Build 92%     🏆 传奇进度 87%
```

## 🏠 首页总览

### Insight Screen（第一屏）

分析完成后首先进入惊喜结果页：

- **总资产价值** — 你账号所有物品的估值总和
- **隐藏财富** — 未充分利用的资产
- **Build 就绪度** — 你最接近的 meta Build
- **传奇进度** — 最接近完成的传奇

点击 **"Continue to Action Center"** 进入主界面。

### Action Center（行动中心）

系统根据你的账号数据，按优先级排列今日行动：

| 优先级 | 颜色 | 含义 | 示例 |
|--------|------|------|------|
| **P0** | 🟠 关键 | 最高价值，立刻行动 | 卖 Mystic Coin → +320g |
| **P1** | 🟢 成长 | 重要但不紧急 | 打 T4 碎层 → +120g/h |
| **P2** | 🔵 可选 | 锦上添花 | 完成日常 → ~10g |

每条行动附带 **评分拆解**（gold / build / time / risk），让你知道为什么它被推荐。

### 策略选择器

在 Action Center 顶部可切换优化策略：

| 策略 | 效果 |
|------|------|
| **Balanced**（默认） | 均衡推荐 |
| **Gold Farming** | 优先赚钱行动 |
| **Build Completion** | 优先组 Build |
| **Legendary Rush** | 优先传奇进度 |
