🔑 How GW2 API Keys Work

What is an API Key?
An API key is a token from ArenaNet that grants read-only access to your GW2 account data.

Is it safe?
✅ Read-only — cannot trade, delete items, or change your account
✅ Session-only — your key is never stored on our servers by default
✅ Revocable — delete the key anytime from ArenaNet

How to get one:

  1. Visit account.arena.net/applications
  2. Click "Create New Key"
  3. Select permissions (recommended: account, characters, inventories, wallet, tradingpost, builds, progression, unlocks)
  4. Copy the key and paste it above

🔒 Your ArenaNet password is NEVER requested. Only the API key.
